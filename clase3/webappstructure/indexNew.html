Ejercicio 0: Descargar e Instalar OpenJDK, Apache Maven, Payara Server y Apache Netbeans (Crear variables de ambiente JAVA_HOME y MAVEN_HOME)

  • 0.1 - Descargar e Instalar OpenJDK

    Si usted ya tiene instalado el OpenJDK 11 LTS omita éste paso. Ir la página de AdoptOpenJDK y Descargar el instalador correspondiente a su sistema Operativo:

    El Link de descarga Directa: (download)



  • 0.2 - Crear Variable de ambiente JAVA_HOME

    En Windows: Presionar tecla de Windows, escribir This PC (Esta PC) y seguir la siguiente secuencia de pasos:





    Presionar OK, tres veces y luego verificar abriendo una terminal del sistema Operativo y escribir los siguientes comandos:

    • java -version

    • javac -version

    • jar

    Sí la respuesta es como la que se muestra en las capturas de pantalla su Variable JAVA_HOME está correctamente configurada

  • 0.3 - Descargar e Instalar Apache Maven

    Si usted ya tiene instalado y configurado Apache Maven omita éste paso. Ir a la página de Apache Maven y descargar el archivo zip o tar.gz correspondiente:

    El proceso de instalación es simplemente extraer el archivo zip o tar.gz en un Path definido por ejemplo C:\WorkNew\

  • 0.4 - Crear Variable de ambiente MAVEN_HOME

    De manera similar a la variable JAVA_HOME crear la variable de ambiente MAVEN_HOME, pero apuntando al directorio donde esta extraido el archivo de Apache Maven.

    Por ejemplo: C:\WorkNew\Tools\apache-maven-3.6.3

    Tambien agregar %MAVEN_HOME%\bin a la variable Path del Sistema Operativo, vea las siguientes capturas de pantalla:

    Presionar OK, tres veces y luego verificar abriendo una terminal del sistema Operativo y escribir el siguiente comando:

    • mvn -version

    Sí la respuesta es como la que se muestra en las capturas de pantalla su Variable MAVEN_HOME está correctamente configurada

  • 0.5 - Descargar e Instalar Payara Server

    Si usted ya tiene instalado y configurado Payara Server omita éste paso. Ir a la página de Payara Server y descargar el archivo zip correspondiente:

    Link de descarga directa: (download)

    Se puede crear una variable de ambiente llamada PAYARA_HOME apuntando al directorio donde se descomprimió el servidor, y Agregar %PAYARA_HOME%\bin a la variable Path para poder iniciar el servidor desde linea de comandos.

    • Para iniciar el Payara server desde linea de comandos ejecutar el siguiente comando: asadmin start-domain domain1

  • 0.6 - Descargar e Instalar Apache NetBeans

    Si usted ya tiene instalado y configurado Apache Netbeans omita éste paso. Ir a la página de Apache Netbeans y descargar el archivo zip o Ejecutable correspondiente:

    Link de descarga directa: (download)
